# Pedlar Street - Premium Ceylon Cinnamon & Spices

Official website for Pedlar Street Trading Company, a leading exporter of premium Ceylon cinnamon and spices from Sri Lanka.

## Website Features

- **Product Showcase** - Premium Ceylon cinnamon quills, ground powder, and organic spice blends
- **International Export** - Bulk orders with flexible packaging for global markets
- **Quality Certifications** - ISO 22000, HACCP, and FDA compliant products
- **Contact & Inquiry** - Easy-to-use contact form for wholesale inquiries
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices

## Hosted on

GitHub Pages: [pedlarstreet.com](https://pedlarstreet.com)

## Quick Start

1. Clone the repository
   ```bash
   git clone https://github.com/YOUR_USERNAME/PedlarStreet.git
   cd PedlarStreet
   ```

2. Open `index.html` in your browser to view locally

3. Make changes and commit
   ```bash
   git add .
   git commit -m "Your message"
   git push
   ```

## Deployment

This site is deployed using GitHub Pages. Simply push to the `main` branch and changes will be live automatically.
